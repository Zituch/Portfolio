Project 1 - Automation Bot

Overview of the automation bot for Dota 2 custom games.

Introduction

This project demonstrates the automation of various tasks in a Dota 2 custom game using Python. The bot automates tasks such as hero selection, skill use, and battle engagement.

Features

  • Automated hero selection
  • Skill usage automation
  • Continuous battle engagement

# Example code snippet
def main_loop():
    global stop_loop

    while True:
        countdown(5)  # 5-second countdown before each iteration
        print("Starting sequence...")
        perform_actions()  # Pre-battle setup

        print("Entering battle phase...")
        battle_finished = press_key_during_battle('1', duration=600, check_interval=2)

        if battle_finished:
            click_confirm_button()

        if stop_loop:
            print("Stop signal received. Exiting...")
            break
