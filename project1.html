Project 1 - Automation Bot

Overview of the automation bot for Dota 2 custom games.

Introduction

This project demonstrates the automation of various tasks in a Dota 2 custom game using Python. The bot automates tasks such as hero selection, skill use, and battle engagement.

Features

  • Automated hero selection
  • Skill usage automation
  • Continuous battle engagement
  • Easy setup and configuration
